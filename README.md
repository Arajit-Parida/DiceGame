# 🎲 Dice Game

A simple and interactive Dice Game built using **React.js**, **JavaScript**, and **CSS**. This project allows users to roll a dice, select numbers, track scores, and learn game rules through a clean and responsive user interface.

## 🚀 Features

* 🎯 Select a number before rolling the dice
* 🎲 Random dice roll generation
* 📈 Real-time score tracking
* ⚠️ Score deduction for incorrect guesses
* 🔄 Reset score functionality
* 📖 Show/Hide game rules
* 📱 Responsive and user-friendly design

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* CSS3
* Styled Components

## 🎮 How to Play

1. Select a number from **1 to 6**.
2. Click on the dice image to roll the dice.
3. If your selected number matches the dice number:

   * Your score increases by the selected number.
4. If the number does not match:

   * 2 points are deducted from your score.
5. Continue playing to achieve the highest score possible.

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/Arajit-Parida/DiceGame.git
```

### Navigate to Project Folder

```bash
cd DiceGame
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

The application will start on:

```bash
http://localhost:5173
```
